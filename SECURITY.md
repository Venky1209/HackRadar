# Security Policy

## Supported Versions

Only the latest main branch is supported.

## Reporting a Vulnerability

If you find a security issue in HackRadar, do not open a public issue.

Use GitHub's private vulnerability reporting or a private security advisory instead.

Include:

- A short summary of the issue
- The affected route or file
- Reproduction steps
- Any proof-of-concept details that help confirm the problem

We will review reports as soon as possible and coordinate a fix before public disclosure.
